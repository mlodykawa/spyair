Important notes

This version is strong for prototyping, but still heuristic-based.

Is this a precise military detector? No. It uses first-pass rules, watchlist patterns, and movement context. It is useful for spotting likely activity, not as a final ground-truth classifier.
What should come next after this? The next real milestone is moving API calls behind a backend or Cloudflare Worker, then adding historical storage, better watchlists, and persistent alerts.
Why is this already valuable? Because you now have live data, filtering, regional switching, watchlists, and an event log in one working interface.

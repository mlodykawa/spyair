Important notes

This version is solid for prototyping, but still heuristic-based.

Is this a precise military detector? No. It is a first-pass classifier based on callsign patterns and watchlist logic.
Why does fallback mode exist? Because live upstream feeds can fail. Fallback prevents the UI from looking dead while you keep building.
What should come next? Server-side filtering, persistent history, stronger watchlists, and backend alerting.

Important notes

Before we build on this, there are two practical limits to know.

Will this show only military aircraft? Not yet. OpenSky returns aircraft state vectors; military filtering has to be added by our own logic on top.
Can this be used commercially as-is? Not safely. OpenSky’s terms say for-profit use requires written permission, so this version is best for testing and prototyping first.
What if the feed fails or gets rate-limited? Then the next version should move the API call behind a Cloudflare Worker proxy with caching and retries.
